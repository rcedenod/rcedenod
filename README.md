# Rafael Cedeño

Ingeniero de Computación · Desarrollador Full-Stack

Egresado de la Universidad Rafael Urdaneta (URU), Maracaibo, Venezuela.  
Me apasiona la arquitectura de software, el diseño de sistemas y construir soluciones escalables y bien planificadas.  
Trabajo en todo el stack — backend, frontend y despliegue — y me importa el rendimiento, el código limpio y hacer las cosas bien.

---

## Qué hago

- **Backend** — APIs REST, Sockets, WebSockets, gRPC, autenticación, seguridad, modelado de bases de datos
- **Frontend** — Interfaces web y móviles con React y React Native
- **Despliegue e infraestructura** — Servidores Linux, Docker, Nginx, PM2, plataformas cloud
